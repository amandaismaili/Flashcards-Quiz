# Flashcards Quiz

A command-line flashcard quiz application built in Python for studying and self-assessment. Users can create flashcard decks, run quiz sessions, and track scores.

## Features

- Create and store flashcard question/answer pairs
- Run quiz sessions through stored cards
- Score tracking per session
- Data saved in JSON files (`questions.json`, `quizdata.json`)

## How to Run

```bash
python flashcards.py
```

No external dependencies — Python 3 only.

## Concepts Demonstrated

- JSON file I/O
- CLI interaction loop
- Basic quiz/scoring logic in Python
